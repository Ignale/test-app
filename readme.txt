После распаковки содержимого репозитория запускаем само приложение npm start,
Обязательно запускаем сервер на 5000 localhost такой командой 
json-server db.json -m ./node_modules/json-server-auth --port 5000
для того, чтобы запустить его с модулем авторизации. 

При запуске можно зарегестрироваться, или ввести уже готовый логин и пароль
логин: ignat.lex@gmail.com
пароль: 123456

Во вкладке контакты можно создавать, удалять, редактировать, фильтровать список контактов.
Фильтрация осуществляется не только по имени, но и по специальности, без учета регистра.
